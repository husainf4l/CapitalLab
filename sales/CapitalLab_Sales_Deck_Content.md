## SLIDE 13 — ANALYZER INTEGRATION

**Headline:** Works with the Equipment You Already Have

**Supported import formats:**
| Format | Protocol | Common Analyzers |
|---|---|---|
| HL7 v2.x | Industry standard | Beckman Coulter, Siemens |
| ASTM E1381 | LIS interface | Mindray, Sysmex, Abbott |
| CSV | Universal | Any analyzer with export |
| XML | Structured data | Roche, bioMérieux |

**How it works:**
1. Analyzer exports results file (auto or manual)
2. Lab technician uploads file in Capital Lab
3. System parses and validates all results
4. Results matched to patient orders automatically
5. Abnormal values flagged for review
6. Doctor reviews and approves

**No middleware required. No additional software licenses.**
